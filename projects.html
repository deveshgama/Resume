Tic Tac Toe

Project Overview :

  • The Tic Tac Toe project is a web-based implementation of the classic two-player game, designed to be simple, interactive, and user-friendly.
  • The game allows users to play in two modes: Single Player (against the computer) and Multiplayer (against another player).

    • Key Features :

  • Single Player and Multiplayer modes
  • Interactive game board with hover effects, each with a 'data-index' for indentification
  • Game status display and reset button
  • Computer AI for Single Player mode
  • A status message is displayed to guide players, such as prompting them to select a mode or indicating the current game state.
  • Winning combinations detection and game result display

    • User Experience :

  • The game is designed to be intuitive, with clear instructions and interactive elements to enhance player engagement.
  • The responsive design ensures compatibility across various devices, providing a seamless gaming experience.

    • Objective :

  • The project aims to showcase my skills in front-end development, including HTML, CSS, and JavaScript, as well as user interface design and game logic implementation.
